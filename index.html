Navigation Bottom

How do i create bottom navigation in mobile interface by using styles from CusMeDroid?

Versi 1


Follow the code below : 
<!DOCTYPE html>
<html>
    <head>
        <title>Navigation Bottom | CusMeDroid
        <link rel='stylesheet' type='text/css' href='https://cusmedroid.github.io/css/style.css'>
    </head>
    <body>
        <div class="header animate-bottom">
            <div class="" align="center">
                <div class="column-header"><a href="#" title="Home" class="bbtn"><i class="fa fa-home" aria-hidden="true"></i></a></div>
                <div class="column-header"><a href="#collection" title="Collection" class="bbtn"><i class="fa fa-picture-o" aria-hidden="true"></i></a></div>
                <div class="column-header"><a href="#like" title="Like" class="bbtn"><i class="fa fa-heart" aria-hidden="true"></i></a></div>
                <div class="column-header" id='musicp2'><a title="Play" class="bbtn"><i class="fa fa-play" aria-hidden="true"></i></a></div>
                <div class="column-header" id='musicpa2'><a title="Pause" class="bbtn"><i class="fa fa-pause" aria-hidden="true"></i></a></div>
            </div>
        </div>
    </body>
</html>

Versi 2


Follow the code below : 
<!DOCTYPE html>
<html>
    <head>
        <title>Navigation Bottom | CusMeDroid
        <link rel='stylesheet' type='text/css' href='https://cusmedroid.github.io/css/style.css'>
    </head>
    <body>
        <div class='header animate-bottom'>
            <div class='me-bottom-bar' align='center'>
                <a onclick='meHome()' title='Home' class='me-menu-item me-current'><i class='fa fa-home' aria-hidden='true'></i></a>
                <a onclick='meCollect()' title='Collection' class='me-menu-item'><i class='fa fa-picture-o' aria-hidden='true' style='margin-left: -1px;'></i></a>
                <a onclick='meLike()' title='Like' class='me-menu-item'><i class='fa fa-heart' aria-hidden='true' style='margin-left: -1px;'></i></a>
                <a class='me-nav-indicator'></a>
                <a title='Play/Pause' class='me-menu-item'>
                    <i class='fa fa-play' aria-hidden='true' id='musicp2'></i>
                    <i class='fa fa-pause' aria-hidden='true' style='margin-right: -2px;' id='musicpa2'></i>
                </a>
            </div>
        </div>
    </body>
</html>

Note

With a cusmedroid stylesheet, it makes it very easy for you to pick up some style features.
And you can specify an action that works for moving a page with the href in tag a or you can use the id with a click listener or you can also add onclick using the javascript language.

If you want the navigation bottom view on the desktop delete the header class.
